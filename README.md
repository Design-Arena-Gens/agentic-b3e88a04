## Agent Factory MCQ Bank

Interactive study companion for the presentation “Agent Factory: Building Digital Full-Time Equivalents (FTEs)”.

### Features

- 40+ curated multiple-choice questions organized by thematic sections.
- Difficulty filtering, instant scoring, and per-question explanations.
- Search across the entire bank to focus on specific topics or terminology.

### Local Development

```bash
npm install
npm run dev
```

Navigate to `http://localhost:3000` to use the experience.

Run a production build with:

```bash
npm run build
npm start
```

### Deployment

Designed for Vercel. Build artifacts are generated via `npm run build`. Use `vercel deploy --prod` with the provided `VERCEL_TOKEN`.
